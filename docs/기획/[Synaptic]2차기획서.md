# 개념 연결 학습 플랫폼 2차 기획서

## 1. 서비스 개요

### 비전
- "지식을 연결하여 더 깊은 이해를 만들다" - 뇌의 시냅스처럼 개념과 개념을 연결하는 학습 경험
- 단순 암기가 아닌, 개념 간 연결을 통한 직관적 이해와 장기 기억 형성
- 시각적 개념 연결을 통한 새로운 통찰력 발견
- 개인화된 지식 네트워크 구축과 공유

### 핵심 가치
1. **연결성** - 파편화된 지식을 유기적으로 연결하여 통합적 이해 도모
2. **직관성** - 시각적 시냅스 맵을 통한 개념 구조의 직관적 파악
3. **확장성** - 지식 노드 간 자연스러운 확장과 심화 학습
4. **지속성** - 연결 기반 학습을 통한 장기 기억 형성
5. **공유성** - 개인의 지식 네트워크를 통한 집단 지성 실현

## 2. 핵심 기능

### A. 시냅스 맵 시스템

1. **개념 노드 생성**
   - 핵심 개념 노드 생성
   - 노드 타입 분류 (이론, 구현, 예시, 질문 등)
   - 노드별 이해도/숙련도 표시
   - 태그 및 메타데이터 관리

2. **개념 연결 관리**
   - 노드 간 연결 생성 및 관리
   - 연결 강도 및 유형 설정
   - 연결 이유 및 설명 추가
   - 양방향 참조 시스템

3. **시각화 시스템**
   - 뉴런 형태의 시각화
   - 연결 강도에 따른 시각적 표현
   - 확장/축소 가능한 인터랙티브 뷰
   - 다양한 시각화 모드 (neural, mindmap, hierarchy)

### B. 학습 노트 시스템

1. **구조화된 노트 작성**
   - 개념 노드와 직접 연결
   - 마크다운 기반 작성
   - 코드 블록 및 예시 포함
   - 실시간 시냅스 맵 연동

2. **노트 템플릿**
   - 개념 정의 템플릿
   - 구현 예시 템플릿
   - 면접 질문 템플릿
   - 문제 해결 템플릿

### C. 지식 확장 시스템

1. **연결 기반 학습**
   - 관련 개념 자동 추천
   - 지식 확장 경로 제안
   - 개념 간 관계 탐색
   - 학습 순서 최적화

2. **복습 최적화**
   - 연결 기반 복습 추천
   - 개념 네트워크 강화 학습
   - 이해도에 따른 복습 주기 조정
   - 연관 개념 통합 복습

### D. 블로그 시스템

1. **시냅스 맵 연동 블로그**
   - 시냅스 맵 스냅샷 포함
   - 관련 개념 자동 링크
   - 개념 네트워크 시각화
   - 독자의 이해도 경로 제시

2. **지식 공유 최적화**
   - SEO 최적화
   - 소셜 공유 기능
   - 개념 맵 임베딩
   - 독자 피드백 시스템

## 3. 사용자 경험 (UX) 설계

### A. 초기 사용자 경험
1. **온보딩 프로세스**
   - 시냅스 맵 사용법 튜토리얼
   - 첫 개념 노드 생성 가이드
   - 연결 생성 실습
   - 시각화 도구 활용법

2. **초기 지식 구조화**
   - 관심 분야 선택
   - 핵심 개념 노드 생성
   - 기본 연결 구성
   - 확장 방향 제안

### B. 학습 경험
1. **개념 탐색**
   - 관심 개념 검색
   - 연결된 개념 탐색
   - 학습 경로 발견
   - 지식 맵 확장

2. **지식 구조화**
   - 새로운 연결 생성
   - 기존 연결 강화
   - 개념 관계 정의
   - 이해도 업데이트

## 4. 기술 구현 계획

### A. 프론트엔드
- React + TypeScript 기반
- 시냅스 맵 시각화 (D3.js/Three.js)
- 실시간 협업 지원
- 반응형 디자인

### B. 백엔드
- Node.js + NestJS
- 그래프 데이터베이스 활용
- 실시간 동기화
- 확장 가능한 API 구조
